# k6 가이드

## k6란

Go로 만든 부하 테스트 도구. JavaScript로 시나리오를 작성하고 CLI로 실행한다.

---

## 핵심 개념

### VU (Virtual User)

동시에 요청을 보내는 가상 사용자. VU 1명은 아래 루프를 반복한다:

```
요청 → 응답 대기 → 응답 수신 → 즉시 다음 요청 → 반복
```

### Iteration

VU 1명이 `export default function()`을 1번 실행하는 것 = 1 iteration.

### Executor

| Executor | 설명 | 사용 시점 |
|----------|------|----------|
| `constant-vus` | VU 수 고정 | 안정 상태 테스트 |
| `ramping-vus` | VU 단계적 증감 | spike 테스트 |
| `constant-arrival-rate` | RPS 고정, VU 자동 조절 | 목표 RPS 테스트 |
| `ramping-arrival-rate` | RPS 단계적 증감 | RPS 기반 점진적 부하 |
| `per-vu-iterations` | VU당 N회 실행 | 정확히 N번 |
| `shared-iterations` | 전체 N회를 VU가 나눠 실행 | 총 N건 빠르게 처리 |

### Stages

```javascript
export const options = {
  stages: [
    { duration: '2m', target: 100 },   // ramp-up
    { duration: '5m', target: 100 },   // steady
    { duration: '1m', target: 500 },   // spike
    { duration: '3m', target: 500 },   // 유지
    { duration: '2m', target: 0 },     // ramp-down
  ],
};
```

## 여러 URL 동시 호출

### scenarios (추천)

각 시나리오가 독립된 VU, duration, executor를 가진다.

```javascript
export const options = {
  scenarios: {
    fast: { executor: 'constant-vus', vus: 150, duration: '5m', exec: 'callFast' },
    slow: { executor: 'constant-vus', vus: 50, duration: '5m', exec: 'callSlow' },
  },
};

export function callFast() {
  http.get('http://bff/mock/delay/100ms/size/1kb');
}
export function callSlow() {
  http.get('http://bff/mock/delay/2s/size/1mb');
}
```

### 랜덤/가중치

```javascript
export default function () {
  const rand = Math.random();
  if (rand < 0.7) http.get('.../delay/100ms/size/1kb');      // 70%
  else if (rand < 0.9) http.get('.../delay/500ms/size/100kb'); // 20%
  else http.get('.../delay/2s/size/1mb');                      // 10%
}
```

### group (메트릭 분리)

```javascript
import { group } from 'k6';
export default function () {
  group('fast', () => { http.get('.../delay/100ms/size/1kb'); });
  group('slow', () => { http.get('.../delay/2s/size/1mb'); });
}
```

| 방법 | VU 독립 제어 | 메트릭 분리 | 비율 제어 |
|------|:-----------:|:---------:|:--------:|
| scenarios | O | O | O (VU 수) |
| 랜덤/가중치 | X | X | O (확률) |
| group | X | O | X |

## 환경변수

```bash
k6 run -e DELAY=2s -e SIZE=1mb -e VUS=500 scenario.js
```

```javascript
const DELAY = __ENV.DELAY || '100ms';
const SIZE = __ENV.SIZE || '1kb';
```

## Checks

```javascript
const res = http.get('http://bff/mock/delay/100ms/size/1kb');
check(res, {
  'status 200': (r) => r.status === 200,
  'latency < 500ms': (r) => r.timings.duration < 500,
});
```

## Tags & Custom Metrics

```javascript
import { Trend } from 'k6/metrics';
const slowLatency = new Trend('slow_api_latency');

export function callSlow() {
  const res = http.get('.../delay/2s/size/1mb', { tags: { type: 'slow' } });
  slowLatency.add(res.timings.duration);
}
```

## 결과 출력

```bash
k6 run test.js                                    # CLI 요약
k6 run --out json=results.json test.js             # JSON
k6 run --out csv=results.csv test.js               # CSV
k6 run --out experimental-prometheus-rw \           # Prometheus
  --env K6_PROMETHEUS_RW_SERVER_URL=http://prometheus:9090/api/v1/write test.js
```

---

## Setup / Teardown

```javascript
export function setup() {
  return { startTime: Date.now() };
}
export default function (data) {
  http.get('http://bff/mock/delay/100ms/size/1kb');
}
export function teardown(data) {
  console.log(`Duration: ${Date.now() - data.startTime}ms`);
}
```

## CLI 옵션

```bash
k6 run --vus 500 --duration 10m test.js    # VU/duration 오버라이드
k6 run -e DELAY=2s test.js                 # 환경변수
k6 run --http-debug test.js                # 요청/응답 상세 출력
k6 run --no-summary --out json=r.json test.js  # summary 끄고 JSON만
```
